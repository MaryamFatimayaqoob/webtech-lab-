# Lab 05: Student Management System

A complete student management system built with PHP and MySQL featuring full CRUD operations, authentication, and dashboard analytics.



## ✨ Features Implemented

### Core Features
- ✅ **Student Registration** - Add new students with personal, academic, and guardian information
- ✅ **View All Students** - Display students in sortable table with search/filter
- ✅ **Student Profiles** - Detailed view of individual student records
- ✅ **Edit Records** - Update existing student information
- ✅ **Delete Records** - Remove student records with confirmation
- ✅ **Search & Filter** - Search by name, ID, course, and filter by status

### System Features
- 🔐 **Authentication System** - Secure login/logout with session management
- 📊 **Dashboard Analytics** - Statistics cards showing total students, active count, average GPA
- 📈 **Course Distribution** - Visual breakdown of students by course
- 👤 **User Profile** - Account management page
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile

### Database Features
- 🗄️ **MySQL Database** with 3 tables (students, users, courses)
- 🔗 **PDO Connections** for secure database interaction
- 🛡️ **SQL Injection Prevention** using prepared statements

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| PHP 8.4+ | Backend logic & authentication |
| MySQL 8.0 | Database management |
| HTML5 | Structure |
| CSS3 | Styling & responsive design |
| PDO | Database connection |
